---
title: 'Get All Trips'
api: 'GET https://api3.cloudsafaris.com/v1/get-all-trips'
---

### Request Body Parameters

<ParamField body="limit" type="number" default={200}>
  The limit on the number of trips to return.
</ParamField>

<ParamField body="offset" type="number" default={0}>
    If requesting another set of trips, this will apply the offset to the next set of trips.
</ParamField>

<ParamField body="country" type="string[]" default="none">
    An array of string values of either `TZ` or `KE`, or both.
</ParamField>

<RequestExample>
  ```json
  {
    header: Auth
  }
  ```
</RequestExample>


### Response Body Parameters
<ResponseField name="id" type="number">
  The trip id is the unique id of the trip returned.
</ResponseField>

<ResponseExample>
```json
{ "status": "success" }
```
</ResponseExample>

<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      The full name of the user
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Whether the user is over 21 years old
    </ResponseField>
  </Expandable>
</ResponseField>
