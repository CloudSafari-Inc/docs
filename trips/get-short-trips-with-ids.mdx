---
title: 'Get Short Trips with IDs'
api: 'POST https://api3.cloudsafaris.com/v1/trips/get-short-trips-with-ids'
---

### Header Parameters
<ParamField header="CloudSafaris-API-Version" type="string" default="v1">
  Currently the only acceptable value is `v1`.
</ParamField>

### Request Body Parameters

<ParamField body="trip_ids" type="number[]" required>
  The unique identifiers for the trips that you expect to receive. Maximum of `100` trip ids to enter here.
</ParamField>

<RequestExample>
  ```json
  {
    "trip_ids": [5,1,20]
  }
```
</RequestExample>

### Response Body Parameters
The fields returned are the same as those in the data from the [Get All Short Trips Response](/trips/get-all-short-trips#response-body-parameters) endpoint, with the exception that it does not return an offset number.
