---
title: 'Get Trip With ID'
api: 'POST https://api3.cloudsafaris.com/v1/trips/get-trip-with-id'
---

### Header Parameters
<ParamField header="CloudSafaris-API-Version" type="string" default="v1">
  Currently the only acceptable value is `v1`.
</ParamField>

### Request Body Parameters

<ParamField body="trip_id" type="number">
  The unique identifier for the trip.
</ParamField>

<RequestExample>
  ```json
  {
    "trip_id": 1
  }
```
</RequestExample>

### Response Body Parameters
The fields returned are the same as those in the data from the [Get All Trips Response](/trips/get-all-trips#response-body-parameters) endpoint, with the exception that it does not return an offset number or an array of trip objects, but just a singular trip object.
