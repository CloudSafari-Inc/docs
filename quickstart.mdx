### Create an Account

To create an account, please head over to the following link:

https://simbas.ai/register?signup

Follow through the sign-up process where you'll be prompted to create a personal profile, and a profile for your company.

### Logging In

If you've already created an account, go to the following link to login:

https://simbas.ai/register

## Navigation

After creating an account (and following through with the onboarding) or logging in, you'll be redirected to the dashboard where you'll see the following navigation menu:

<AccordionGroup>
  <Accordion title="Company">
    When you select "Company" you'll see a pop-up where you can edit the information around your company.
  </Accordion>
  <Accordion title="Profile">
    When you select on "Profile" you'll see a pop-up where you can edit the information around your personal profile for Simbas.AI
  </Accordion>
  <Accordion title="AI Chat">
    "AI Chat" is one of the core features for Simbas.AI. This is the ChatGPT-like interface where you can interact with all the uploaded data, check for availability, and more.
  </Accordion>
  <Accordion title="Templates">
    Templates are the stored notes that you can create to allow for quick access and structured responses from your team within the AI Chat notepad.
  </Accordion>
  <Accordion title="Supplier Pricing Data">
    Supplier Pricing Data is a set of stored pricing information regarding any data that you want accessible via the AI chat. This is also a BETA feature for the Dynamic Itineraries.
  </Accordion>
  <Accordion title="Itineraries">
    Clicking on "Itineraries" will lead you to the page where you can create an itinerary from scratch, or pre-generate from AI, and where you can view all saved itineraries.
  </Accordion>
</AccordionGroup>
