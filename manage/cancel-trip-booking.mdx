---
title: 'Cancel Trip'
api: 'POST https://api3.cloudsafaris.com/v1/manage/cancel-trip'
---

If you need to cancel a trip for any reason, you can do so by sending the following parameters to our endpoint.

### Header Parameters
<ParamField header="CloudSafaris-API-Version" type="string" default="v1">
  Currently the only acceptable value is `v1`.
</ParamField>


### Request Parameters
<ParamField body="booking_uuid" type="uuid">
    The uuid (unique id) associated with the booking.
</ParamField>
<ParamField body="reason" type="string">
    The reason why this trip was cancelled.
</ParamField>

<RequestExample>
```json
{
    "booking_uuid": "123e4567-e89b-12d3-a456-426614174000",
    "reason": "Traveller stopped responding to the messages, it's been 1 week."
}
```
</RequestExample>


### Response Parameters
<ResponseField name="success" type="boolean">
    Whether the trip was cancelled successfully or not.
</ResponseField>
<ResponseField name="message" type="boolean">
    A message associated with this request, if any.
</ResponseField>


<ResponseExample>
```json
{
    "success": true,
    "message": "Trip was successfully cancelled."
}
```
</ResponseExample>
