---
title: 'Get Webhooks'
api: 'POST https://api3.cloudsafaris.com/v1/webhooks/get-webhooks'
---

<Info>
This endpoint allows you to create new webhooks for your organization. 
</Info>

### Header Parameters
<ParamField header="CloudSafaris-API-Version" type="string" default="v1">
  Currently the only acceptable value is `v1`.
</ParamField>

### Request Body Parameters

<ParamField body="platform_id" type="uuid" required>
    The unique identifier for your platform, v4 UUID.
</ParamField>



<RequestExample>
  ```json
{
    "platform_id": "00000000-0000-0000-0000-000000000000"
}
```
</RequestExample>


### Response Body Parameters

<ResponseField name="id" type="number">
    The unique identifier created in our database for your webhook 
</ResponseField>
<ResponseField name="platform_id" type="uuid">
    The unique identifier for your platform, v4 UUID.
</ResponseField>
<ResponseField name="url" type="url">
    The url that you would like us to send updates to. 
</ResponseField>
<ResponseField name="event_type" type="string">
    An enum specifying the type of webhook you are creating. Options are: `["STATUS_UPDATE", "TRIP_MESSAGE"]`
</ResponseField>
<ResponseField name="is_active" type="boolean">
    Specify whether this webhook is currently active or not.  
</ResponseField>
<ResponseField name="created_at" type="date">
    The date which the webhook was created, returned in in ISO 8601 format
</ResponseField>
<ResponseField name="updated_at" type="date">
    The date which the webhook was last updated, returned in in ISO 8601 format
</ResponseField>
<ResponseExample>
```json
[
    {   
        "id": 123
        "platform_id": "00000000-0000-0000-0000-000000000000",
        "url": "https://yourendpointurl.com"
        "event_type": "STATUS_UPDATE", 
        "is_active": true,
        "created_at": "2024-04-05T17:56:19.014+00Z", 
        "updated_at": "2024-04-05T17:56:19.014+00Z"
    }, 
    // More webhooks
]
```
</ResponseExample>
